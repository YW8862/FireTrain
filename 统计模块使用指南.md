# 统计模块使用指南

## 快速开始

### 1. 查看个人统计
```bash
curl http://localhost:8000/api/stats/personal
```

### 2. 查看训练趋势（最近 7 天）
```bash
curl "http://localhost:8000/api/stats/trend?days=7"
```

### 3. 查看步骤分析
```bash
curl http://localhost:8000/api/stats/step-analysis
```

### 4. 查看完整统计概览
```bash
curl "http://localhost:8000/api/stats/overview?days=7"
```

## API 参数说明

### /api/stats/trend
- `days` (可选): 查询天数，默认 7 天，范围 1-30 天
- 示例：`/api/stats/trend?days=15`

### /api/stats/overview
- `days` (可选): 趋势图天数，默认 7 天，范围 1-30 天
- 示例：`/api/stats/overview?days=30`

## 前端集成示例

### Vue 3 + ECharts 示例

```vue
<template>
  <div>
    <!-- 个人统计卡片 -->
    <el-card v-if="personalStats">
      <div class="stat-item">
        <span class="label">总训练次数:</span>
        <span class="value">{{ personalStats.total_trainings }}</span>
      </div>
      <div class="stat-item">
        <span class="label">平均分:</span>
        <span class="value">{{ personalStats.average_score }}</span>
      </div>
      <div class="stat-item">
        <span class="label">最佳成绩:</span>
        <span class="value">{{ personalStats.best_score }}</span>
      </div>
    </el-card>

    <!-- 趋势图表 -->
    <div ref="trendChart" style="width: 100%; height: 400px;"></div>

    <!-- 步骤分析图表 -->
    <div ref="stepChart" style="width: 100%; height: 400px;"></div>
  </div>
</template>

<script setup>
import { ref, onMounted } from 'vue'
import * as echarts from 'echarts'
import request from '@/api/request'

const personalStats = ref(null)
const trendData = ref([])
const stepAnalysis = ref([])

// 获取个人统计
async function fetchPersonalStats() {
  const res = await request.get('/api/stats/personal')
  personalStats.value = res.data
}

// 获取趋势数据
async function fetchTrendData() {
  const res = await request.get('/api/stats/trend?days=7')
  trendData.value = res.data.trend_data
  renderTrendChart()
}

// 获取步骤分析
async function fetchStepAnalysis() {
  const res = await request.get('/api/stats/step-analysis')
  stepAnalysis.value = res.data.step_analysis
  renderStepChart()
}

// 渲染趋势图
function renderTrendChart() {
  const chart = echarts.init(document.querySelector('[ref="trendChart"]'))
  chart.setOption({
    title: { text: '训练趋势' },
    tooltip: { trigger: 'axis' },
    xAxis: {
      type: 'category',
      data: trendData.value.map(item => item.date)
    },
    yAxis: {
      type: 'value',
      name: '分数'
    },
    series: [{
      data: trendData.value.map(item => item.average_score),
      type: 'line',
      name: '平均分数',
      smooth: true
    }]
  })
}

// 渲染步骤分析图
function renderStepChart() {
  const chart = echarts.init(document.querySelector('[ref="stepChart"]'))
  chart.setOption({
    title: { text: '步骤分析' },
    tooltip: { trigger: 'axis' },
    xAxis: {
      type: 'category',
      data: stepAnalysis.value.map(item => item.step_name),
      axisLabel: { rotate: 45 }
    },
    yAxis: {
      type: 'value',
      name: '分数',
      min: 0,
      max: 100
    },
    series: [{
      data: stepAnalysis.value.map(item => item.average_score),
      type: 'bar',
      name: '步骤得分',
      itemStyle: {
        color: (params) => {
          return params.value >= 90 ? '#67C23A' :
                 params.value >= 80 ? '#409EFF' :
                 params.value >= 60 ? '#E6A23C' : '#F56C6C'
        }
      }
    }]
  })
}

onMounted(() => {
  fetchPersonalStats()
  fetchTrendData()
  fetchStepAnalysis()
})
</script>
```

## 数据结构说明

### PersonalStatisticsResponse
```typescript
interface PersonalStatisticsResponse {
  user_id: number
  total_trainings: number          // 总训练次数
  completed_trainings: number      // 已完成次数
  total_training_seconds: string   // 总时长（秒）
  average_score: string            // 平均分
  best_score: string               // 最佳成绩
  last_training_at: string         // ISO 8601 日期时间
}
```

### TrainingTrendItem
```typescript
interface TrainingTrendItem {
  date: string              // YYYY-MM-DD
  training_count: number    // 当天训练次数
  average_score: string     // 当天平均分
  best_score: string | null // 当天最高分
}
```

### StepAnalysisItem
```typescript
interface StepAnalysisItem {
  step_name: string              // 步骤名称
  average_score: string          // 该步骤平均分
  success_rate: string           // 成功率 (0-100)
  improvement_suggestion: string | null // 改进建议
}
```

## 常见问题

### Q: 为什么返回的分数是字符串而不是数字？
A: 为了保持精度，使用 Decimal 类型，序列化为字符串。前端可以使用 `parseFloat()` 或 `Decimal.js` 处理。

### Q: 如何刷新统计数据？
A: 系统会在每次完成训练后自动刷新预计算数据。也可以手动调用 `refresh_statistics()` 方法。

### Q: 趋势图最多支持多少天？
A: 最多支持 30 天，避免查询过多数据影响性能。

### Q: 如何处理没有训练记录的新用户？
A: 接口会返回 404 错误，前端应友好提示"暂无训练数据"。

## 注意事项

1. **性能考虑**: 首次查询可能较慢（实时聚合），后续会使用预计算数据
2. **数据准确性**: 基于已完成的训练记录（status = "done" 或 "completed"）
3. **时区处理**: 所有日期时间使用 UTC，前端需转换为本地时区
4. **错误处理**: 建议前端添加 loading 状态和错误提示
