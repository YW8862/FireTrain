# FireTrain灭火器实操测评系统前端美化需求文档

# 1. 文档概述

## 1.1 文档目的

本文档明确FireTrain灭火器实操测评系统（以下简称“系统”）前端美化的核心需求、视觉规范、页面优化细节、组件要求及落地标准，为前端开发提供清晰、可落地的执行依据，确保美化后界面贴合消防实操培训主题，弱化AI科技感，提升界面美观度、专业性和实操引导性，同时保留系统原有核心功能和技术架构。

## 1.2 项目背景

现有系统基于Vue3+Element Plus+ECharts技术栈开发，核心功能为灭火器实操测评（实时视频录制、动作规范检测、得分统计、历史记录查询等）。当前前端存在以下问题：界面布局通用化，缺乏消防培训场景辨识度；视觉元素（配色、图标、文案）带有明显AI科技感，与消防实操主题贴合度低；交互反馈及视觉呈现未能突出“实操引导”核心，用户体验不够贴合培训场景。本次美化聚焦“去AI感、贴消防主题、提美观度、强实操性”，在不推翻原有系统结构和核心功能的前提下，完成前端视觉及交互优化。

## 核心目标

- 去AI感：删除所有AI相关的视觉元素（如3D姿态骨架、科技感图标）和文案（如“AI姿态识别”“模型加载”），替换为贴合消防实操培训的内容。

- 贴消防主题：以工业级消防培训风格为核心，融入消防元素（灭火器、操作步骤、警示符号），配色、图标、文案均贴合消防安全培训场景。

- 提美观度：统一视觉规范，优化页面布局、组件样式和细节动效，打造整洁、严谨、有辨识度的工业级界面，避免杂乱和通用化。

- 强实操性：突出实操引导，优化实时反馈、步骤提示等元素，让用户聚焦“如何规范操作”，而非“AI如何识别”。

- 保落地性：基于现有技术栈（Vue3+Element Plus+ECharts），所有优化需求均可低成本落地，不新增复杂依赖，不推翻原有功能结构。

## 适用范围

本文档适用于FireTrain系统前端美化的开发、测试及验收工作，覆盖系统所有前端页面（登录页、首页、训练页、历史记录页、统计分析页、个人中心页）及所有交互组件。

# 2. 总体设计原则

## 2.1 风格原则

采用「工业级消防培训风格」，核心关键词：严谨、清晰、实操导向、专业安全，坚决避免以下风格：高饱和霓虹色、玻璃拟态、科技感3D元素、AI相关视觉符号，杜绝“AI味”过重的设计。

## 2.2 功能优先原则

美化需以不影响系统核心功能为前提，优先保留“视频录制、动作检测、得分统计、历史查询”等核心功能，所有视觉和交互优化均为“辅助实操、提升体验”，不新增冗余功能，不遮挡核心操作区域（如视频流、控制按钮）。

## 2.3 一致性原则

全局视觉元素（配色、字体、图标、按钮、间距）保持统一，各页面布局逻辑一致，交互反馈样式统一，避免出现视觉混乱、操作断层的情况。

## 2.4 适配性原则

优化后界面需适配电脑、平板等常用设备，保证视频流、按钮、文字在不同屏幕尺寸下清晰可见、操作便捷，不出现拉伸、错位、文字过小等问题。

# 3. 视觉规范需求（核心，去AI感+贴消防主题）

## 3.1 配色规范（核心优化）

配色以“消防警示+工业专业”为核心，控制颜色数量（不超过5种），避免高饱和碰撞，具体规范如下表：

|颜色类型|色值|使用场景|备注（去AI感要求）|
|---|---|---|---|
|主色（工业深蓝）|#1E40AF|导航栏背景、主操作按钮、图表主色、标题文字|替换原有Element Plus默认蓝色，体现工业专业性，避免科技感蓝|
|辅助色1（消防红）|#D92121|警示提示、错误反馈、关键操作按钮、步骤条当前步骤、边框点缀|低饱和度，避免刺眼，贴合消防警示属性，不用于大面积背景|
|辅助色2（安全绿）|#10B981|成功反馈、步骤条已完成步骤、优秀得分标签、正确操作提示|简洁柔和，体现“规范操作”的安全属性|
|中性灰（浅）|#F8F9FA|页面背景、卡片背景|避免纯色纯白，增加轻微质感，不添加渐变（AI感重）|
|中性灰（中）|#E9ECEF|组件边框、分割线、辅助文字背景|统一边框样式，避免杂乱|
|中性灰（深）|#495057|正文文字、辅助说明文字|保证文字可读性，避免纯黑（过于生硬）|
|禁用色|#C9CDD4|未解锁功能、禁用按钮、未完成步骤|统一禁用样式，视觉区分明显|
补充要求：全局配色不使用任何霓虹色、渐变色（除按钮hover轻微加深外），避免玻璃拟态效果，所有背景以浅灰为基础，不添加科技感纹理。

## 3.2 字体规范

- 字体类型：全局统一使用无衬线字体（Microsoft YaHei/Inter），避免花哨字体，体现工业严谨性。

- 字号分层（统一使用px单位，保证适配性）：

    - 一级标题（系统标题、页面主标题）：20px，粗体（font-weight: 600）；

    - 二级标题（板块标题、卡片标题）：18px，粗体；

    - 三级标题（步骤标题、表单标题）：16px，常规加粗；

    - 正文文字（操作提示、表格内容、说明文字）：14px，常规；

    - 辅助文字（备注、时间、小提示）：12px，常规，颜色为中性灰（深）#495057。

- 文字对齐：标题居中/左对齐（按页面布局），正文左对齐，数字（得分、次数）居中对齐，避免右对齐（不贴合中文阅读习惯）。

## 图标规范（去AI感核心）

### 图标风格

全部使用「扁平化/手绘风消防/安全类图标」，删除所有AI相关图标（如机器人、神经网络、3D点云、科技感箭头），图标风格统一，线条简洁，避免复杂细节。

### 核心图标替换要求

|原有AI感图标|替换后图标（消防/安全类）|使用场景|
|---|---|---|
|AI姿态识别图标|灭火器图标（🧯）|训练页、首页训练入口|
|科技感摄像头图标|简洁摄像头图标（📹）|视频录制区域、训练控制按钮|
|AI分析图标|实操指引图标（如步骤插图缩略图）|训练页指引面板|
|科技感统计图标|图表图标（📊）、星星图标（⭐）|统计页、得分展示|
|科技感记录图标|台账图标（📜）、回放图标（⏯️）|历史记录页、视频回放按钮|
|通用提示图标|警示图标（⚠️）、正确图标（✅）、错误图标（❌）|操作反馈、提示弹窗|
补充要求：图标来源优先选择iconfont免费无版权消防类图标，若需自定义，保持手绘风/扁平化，颜色与全局配色统一（主色、消防红、安全绿）。

## 插图/背景规范

- 插图风格：全部使用「手绘风/扁平化消防实操插图」，避免AI生成的科技感图像、3D图像，插图内容聚焦灭火器操作步骤、安全警示场景。

- 核心插图需求：
        

    - 首页：灭火器6步操作图解（拔销→握管→对准火源→按压→扫射→停止），每步配手绘插图+简短文字说明，错误步骤标消防红，正确步骤标安全绿。

    - 训练页：当前步骤手绘插图（放大展示），如“拔销”“对准火源”等，贴合实时操作指引。

    - 安全须知：消防安全警示插图（如“禁止违规操作”“操作前检查灭火器”），简洁明了。

- 背景要求：页面背景统一为浅灰（#F8F9FA），可添加低透明度（10%）消防纹理（灭火器轮廓、消防栓简笔画），避免纯色纯白、渐变背景、科技感背景。

- 版权要求：所有插图需使用无版权资源（如Canva免费模板、iconfont免费插图），或自定义绘制，确保无版权纠纷。

## 间距规范

严格遵循8px基准间距体系，保证界面整洁有序，避免杂乱，具体要求：

- 页面内边距：统一为24px；

- 板块间距：16px（如首页各板块、统计页各图表之间）；

- 组件间距：8px（如按钮之间、表单项之间）；

- 卡片内边距：24px，卡片之间间距：16px；

- 文字间距：行高1.5倍，段落之间间距16px。

补充：使用Element Plus的Row/Col栅格系统，按上述间距配置，确保各页面布局一致性。

# 4. 各页面详细美化需求

## 4.1 登录页（含注册页）

### 现有问题

表单通用化，无消防主题元素；按钮、边框为Element Plus默认风格，无行业辨识度；无安全提示，AI感无，但缺乏场景感。

### 美化需求

- 布局：居中布局，表单卡片占页面中间区域（宽度500px左右），避免全屏表单，提升聚焦感。

- 背景：浅灰（#F8F9FA）底，添加低透明度（10%）灭火器线描纹理，无其他多余元素。

- 标题：顶部显示“灭火器实操训练测评系统”，字体20px粗体，颜色工业深蓝（#1E40AF），左侧搭配灭火器手绘图标，居中对齐。

- 表单卡片：
        

    - 样式：圆角8px，轻微阴影（box-shadow: 0 2px 12px 0 rgba(0,0,0,0.04)），边框1px消防红（#D92121）细边；

    - 背景：纯白，内边距24px；

    - 表单项：Element Plus Input/Select组件，边框为中性灰（中）#E9ECEF，聚焦时边框变为消防红（#D92121）；placeholder文字贴合场景，如“请输入训练账号”“请输入密码”“请确认密码”。

- 按钮：
        

    - 登录/注册按钮：工业深蓝（#1E40AF）实心按钮，字体14px粗体，hover时背景色加深（#1A3699），无渐变；

    - 切换登录/注册按钮：浅灰描边按钮，字体14px，hover时加消防红描边；

    - 按钮文案：“登录训练系统”“注册训练账号”“切换至登录”“切换至注册”，贴合培训场景。

- 安全提示：表单底部添加浅灰底（#E9ECEF）提示条，文字12px，内容“请使用训练专用账号登录，操作前请阅读安全规范”，左侧搭配警示图标（⚠️）。

## 4.2 首页

### 现有问题

布局通用化，缺乏消防培训场景感；存在AI感文案（如“AI姿态识别”）；无实操引导元素，核心入口不够突出。

### 美化需求

- 布局（分4个区域，贴合消防培训逻辑）：
        

    - 顶部区域：系统标题（同登录页）+ 安全提示条（红底白字，文字“操作前请阅读安全规范，规范操作，注意安全”），右侧添加“安全规范”按钮（浅灰描边，点击弹出弹窗）。

    - 左侧区域（占页面40%）：灭火器操作步骤图解，标题“灭火器规范操作6步”（18px粗体，工业深蓝），下方按步骤展示手绘插图+文字说明，每步标注序号，错误步骤标消防红，正确步骤标安全绿，背景为浅灰（#F8F9FA），边框1px中性灰（中）#E9ECEF。

    - 右侧区域（占页面60%）：快速操作区，背景为纯白，边框1px中性灰（中）#E9ECEF，内边距24px，包含3个核心入口（按钮+图标+文字）：
               

        - 开始训练：工业深蓝实心按钮（大号，宽度100%），搭配灭火器图标，文案“开始实操训练”；

        - 查看历史记录：浅灰描边按钮（宽度100%），搭配台账图标，文案“查看训练记录”；

        - 我的训练统计：浅灰描边按钮（宽度100%），搭配图表图标，文案“我的训练统计”。

    - 底部区域：安全须知，浅灰底（#F8F9FA），边框1px中性灰（中）#E9ECEF，内边距16px，文字14px，内容“1. 训练前请确认摄像头正常；2. 操作过程中请严格按照步骤执行；3. 测评结果仅作为训练参考，不代表实际操作资质”，左侧搭配警示图标（⚠️）。

- 文案优化：删除所有AI感文案，将“AI姿态识别”改为“实操动作规范测评”，“智能训练”改为“实操训练”。

- 细节：所有板块间距16px，按钮间距8px，插图与文字对齐，避免杂乱。

## 4.3 训练页（核心页面，去AI感重点）

### 现有问题

存在AI姿态骨架、AI分析面板等强AI感元素；视频流区域无场景化引导；反馈文案AI感重；步骤条、按钮无消防主题属性；布局未突出实操引导。

### 美化需求

- 布局（核心优化，突出实操引导）：
        

    - 顶部区域：训练标题（18px粗体，工业深蓝），如“灭火器实操训练-第3步：对准火源”，右侧添加训练状态提示（如“未开始”“训练中”“暂停”），下方添加红底白字安全警示条（文字“训练中请保持摄像头对准操作人员，确保动作清晰”）。

    - 中间区域（分左右两栏）：
                

        - 左侧（占70%）：视频流区域，边框为黑色（1px），背景为黑色（无信号/未开启时显示“请开启摄像头”文字，居中，颜色中性灰（深）#495057）；视频流上方添加“当前视角：操作人员全身”提示（12px，浅灰底）；操作错误时，视频流边框变为消防红（2px），同时弹出半透明提示。

        - 右侧（占30%）：实操指引面板（Element Plus Card组件），背景纯白，边框1px中性灰（中）#E9ECEF，内边距24px，包含4个模块（按顺序排列）：
                        

            - 当前步骤插图：手绘风，放大展示（占面板30%），对应当前训练步骤（如“对准火源”）；

            - 操作要求：14px文字，搭配对应图标，如“距离火源2-3米，喷嘴对准火源根部”，文字左对齐；

            - 实时反馈：根据操作情况显示，搭配对应图标（正确✅、警告⚠️、错误❌），颜色对应安全绿/警告黄/消防红，文案为实操导向（如“姿势规范”“需调整喷嘴角度”“操作错误，请重新拔销”），不出现“AI识别”等话术；

            - 得分展示：大号数字（24px粗体，工业深蓝），下方添加等级标签（优秀/合格/待改进），优秀标安全绿，合格标中性灰（深），待改进标消防红。

    - 底部区域：步骤条（Element Plus Steps组件）+ 控制按钮，间距16px；
                

        - 步骤条：宽度100%，已完成步骤标安全绿（配✅图标），当前步骤标消防红（配灭火器图标），未完成步骤标中性灰（中）（配灰色灭火器图标）；步骤文字为14px，简洁明了（如“1. 拔销”“2. 握管”）；

        - 控制按钮（按顺序排列，间距8px）：
                        

            - 开始训练：工业深蓝实心按钮，文案“开始训练”，搭配开始图标；

            - 暂停训练：浅灰描边按钮，文案“暂停训练”，搭配暂停图标；

            - 结束测评：消防红实心按钮，文案“结束测评”，搭配结束图标；

            - 重新训练：浅灰描边按钮，文案“重新训练”，搭配刷新图标。

- 去AI感核心操作：
        

    - 删除右侧“AI姿态分析面板”的3D骨架、关键点可视化，替换为上述实操指引面板；

    - 删除所有AI相关文案，将“AI分析完成”改为“实操测评完成”，“关键点识别错误”改为“操作动作不规范”，“模型加载中”改为“测评系统准备中”；

    - 视频流区域不添加任何科技感边框、动效，仅在操作错误时用红框提示。

- 交互优化：
        

    - 步骤切换时，弹出半透明遮罩（消防红底，透明度50%），显示下一步操作插图+文字，2秒后自动消失；

    - 每步操作完成后，弹出对应反馈提示（Element Plus Message组件），搭配音效（可选，简洁提示音）；

    - 视频加载/系统准备时，使用Element Plus Spin加载组件，搭配文字“测评系统准备中，请稍候”，无AI感加载动效。

## 4.4 历史记录页

### 现有问题

表格通用化，无消防主题区分；反馈文案AI感重；视频回放入口不突出；无操作建议，贴合度低。
