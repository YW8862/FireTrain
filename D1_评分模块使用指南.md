# D1 评分模块使用指南

## 概述

评分模块已实现基于规则 + 固定阈值的模拟评分功能，保证主链路稳定运行。

## 核心功能

### 1. 评分服务（ScoringService）

**文件位置**: `backend/app/services/scoring_service.py`

**主要方法**:
```python
async def score_training(
    self,
    training_type: str,
    video_path: Optional[str] = None,
    frame_data: Optional[List[Dict[str, Any]]] = None,
    duration_seconds: Optional[Decimal] = None
) -> Dict[str, Any]
```

**输入参数**:
- `training_type`: 训练类型（如 `fire_extinguisher`）
- `video_path`: 视频文件路径（可选，当前版本未实际使用）
- `frame_data`: 帧数据索引（可选，为后续 AI 评分预留）
- `duration_seconds`: 实际用时（秒，用于时间评分）

**输出结构**:
```python
{
    "total_score": float,          # 总分（0-100）
    "step_scores": {               # 各步骤分数
        "step1": {
            "step_name": str,      # 步骤名称
            "score": float,        # 步骤分数（0-100）
            "is_correct": bool,    # 是否正确
            "feedback": str,       # 步骤反馈
            "weight": float        # 权重（0-1）
        }
    },
    "feedback": str,               # 总体反馈
    "suggestions": List[str],      # 改进建议列表
    "action_logs": List[Dict],     # 动作日志
    "duration_seconds": float,     # 实际用时
    "time_bonus": float            # 时间奖励/惩罚（可选）
}
```

## 评分规则

### 灭火器操作评分标准

#### 步骤权重配置
```python
STEP_WEIGHTS = {
    "准备阶段": 0.2,      # 准备工作占 20%
    "提灭火器": 0.15,     # 提灭火器占 15%
    "拔保险销": 0.2,      # 拔保险销占 20%
    "握喷管": 0.1,        # 握喷管占 10%
    "瞄准火源": 0.2,      # 瞄准火源占 20%
    "压把手": 0.15,       # 压把手占 15%
}
```

#### 评分阈值
```python
SCORE_THRESHOLDS = {
    "excellent": 90,  # 优秀分数线
    "good": 80,       # 良好分数线
    "pass": 60,       # 合格分数线
}
```

#### 时间评分标准
- **标准时间**: 120 秒（2 分钟）
- **合理时间范围**: 96-144 秒（0.8-1.2 倍）
  - 在范围内：奖励 5 分
  - 超过 1.5 倍（180 秒）：扣 5 分

### 分数段说明

- **90-100 分**: 优秀 - 动作规范，流程熟练
- **80-89 分**: 良好 - 基本掌握操作要领
- **60-79 分**: 合格 - 有提升空间，需加强练习
- **0-59 分**: 不合格 - 操作存在较大问题

## 使用示例

### 1. 在 TrainingService 中调用

```python
from app.services.scoring_service import ScoringService

# 创建评分服务实例
scoring_service = ScoringService()

# 对训练进行评分
result = await scoring_service.score_training(
    training_type="fire_extinguisher",
    duration_seconds=Decimal("125.5")
)

# 获取评分结果
total_score = result["total_score"]
step_scores = result["step_scores"]
feedback = result["feedback"]
suggestions = result["suggestions"]
```

### 3. 直接调用评分服务

```python
from decimal import Decimal
from app.services.scoring_service import ScoringService

async def example():
    scorer = ScoringService()
    
    # 示例 1: 基础评分
    result = await scorer.score_training("fire_extinguisher")
    print(f"总分：{result['total_score']}")
    print(f"反馈：{result['feedback']}")
    
    # 示例 2: 带时间评分
    result = await scorer.score_training(
        training_type="fire_extinguisher",
        duration_seconds=Decimal("100")  # 100 秒完成
    )
    print(f"时间奖励：{result.get('time_bonus', 0)}")
    
    # 示例 3: 分析各步骤
    for step_key, step_data in result["step_scores"].items():
        print(f"{step_data['step_name']}: {step_data['score']}分")
        print(f"  反馈：{step_data['feedback']}")
```

## 与训练模块集成

### 调用流程
```
Training API (complete_training)
  ↓
Training Service (generate_mock_score)
  ↓
Scoring Service (score_training)
  ↓
返回评分结果
  ↓
保存到 Training Record
```

### 数据库落库
评分结果会保存到 `training_records` 表的以下字段：
- `total_score`: 总分
- `step_scores`: 步骤分数（JSON 格式）
- `feedback`: 总体反馈
- `status`: 更新为 "done"
- `completed_at`: 完成时间

## 性能考虑

- **当前实现**: 纯内存计算，无外部依赖，响应时间 < 10ms
- **线程安全**: 无状态设计，支持并发调用
- **缓存策略**: 可根据需要添加结果缓存

## 注意事项

1. **第一版为模拟评分**: 使用随机性和规则组合，不是真实 AI 评分
2. **预留扩展接口**: `video_path` 和 `frame_data` 参数暂未使用
3. **中文注释**: 所有代码均使用中文注释，便于维护
4. **测试覆盖**: 提供完整的测试用例，确保功能稳定
