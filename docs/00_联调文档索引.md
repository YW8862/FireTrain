## 🔍 常见问题快速解决

### 问题 1: Token 鉴权失败 ❌
**症状**: `{"detail": "无法验证凭据", "code": 401}`

**解决**: 
```bash
# 重新登录获取新 token
curl -X POST http://localhost:8000/api/user/login \
  -d "username=student001&password=Test123456"
```

📖 **详情**: [联调记录.md - 常见问题排查](./联调记录.md#问题-1-token-鉴权失败)

---

### 问题 2: 数据库连接失败 ❌
**症状**: SQLAlchemy 连接错误

**解决**:
```bash
# 重启服务
make docker-restart

# 查看日志
docker compose logs mysql
```

📖 **详情**: [联调使用指南.md - 常见问题排查](./联调使用指南.md#问题-2-数据库连接失败)

---

### 问题 3: 视频上传失败 ❌
**症状**: `{"detail": "视频上传失败", "code": "VIDEO_UPLOAD_FAILED"}`

**解决**:
```bash
# 检查权限
ls -la data/videos/
chmod 755 data/videos/
```

📖 **详情**: [联调记录.md - 问题 3](./联调记录.md#问题-3-视频上传失败)

---

### 问题 4: AI 评分超时 ❌
**症状**: `Request timeout after 30000ms`

**解决**: 增加超时时间或检查模型文件

📖 **详情**: [联调记录.md - 问题 4](./联调记录.md#问题-4-ai-评分超时)
