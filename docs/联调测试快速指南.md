## 🔍 常见问题排查

### 问题 1: 无法连接后端

**症状**: `curl: (7) Failed to connect to localhost port 8000`

**解决方案**:
```bash
# 检查服务状态
docker compose ps

# 查看后端日志
docker compose logs backend

# 重启服务
make docker-restart
```

### 问题 2: Token 无效

**症状**: 
```json
{
  "detail": "无法验证凭据",
  "code": 401
}
```

**解决方案**:
1. 确认 Token 未过期（有效期 30 分钟）
2. 重新登录获取新 Token
3. 确认请求头格式正确：`Authorization: Bearer <token>`

### 问题 3: 数据库错误

**症状**: SQLAlchemy 相关错误

**解决方案**:
```bash
# 查看数据库日志
docker compose logs mysql

# 检查数据库连接
docker compose exec backend python3 -c "from app.db.session import engine; print('OK')"
```
