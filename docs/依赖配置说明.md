## 🎯 推荐安装方式

### 方式 1: 自动检测安装（推荐）⭐

```bash
cd backend
./check_env.sh
```

脚本会自动：
- ✅ 检测你的硬件配置
- ✅ 判断是否需要 GPU 版本
- ✅ 清理冲突的包
- ✅ 安装最适合的版本

### 方式 2: 手动安装（CPU 优化版）

```bash
cd backend
source .venv/bin/activate

# 1. 卸载冲突的包
pip uninstall -y opencv-python opencv-contrib-python

# 2. 安装 CPU 版 PyTorch
pip install torch torchvision torchaudio \
  --index-url https://download.pytorch.org/whl/cpu

# 3. 安装其他依赖
pip install -r requirements.txt
```

### 方式 3: 快速安装（使用优化后的 requirements.txt）

```bash
cd backend
source .venv/bin/activate
pip install -r requirements.txt
```

## ⚠️ 常见问题

### 问题 1: OpenCV 版本冲突

**症状**: 
```
ERROR: pip's dependency resolver does not currently take into account all the packages that are installed.
```

**原因**: 同时安装了 `opencv-python`, `opencv-contrib-python`, `opencv-python-headless`

**解决**:
```bash
pip uninstall -y opencv-python opencv-contrib-python
pip install opencv-python-headless==4.9.0.80
```

### 问题 2: PyTorch 安装过大

**症状**: 
```
Installing torch downloaded 2.3 GB
```

**原因**: 安装了 GPU 版本而不是 CPU 版本

**解决**:
```bash
# 卸载当前版本
pip uninstall -y torch

# 安装 CPU 版本
pip install torch --index-url https://download.pytorch.org/whl/cpu
```

### 问题 3: NumPy 版本不兼容

**症状**:
```
ImportError: numpy.core.multiarray failed to import
```

**原因**: NumPy 版本过高或过低

**解决**:
```bash
pip install numpy==1.26.3
```

## 🚀 性能对比

### CPU vs GPU 版本性能

| 任务 | CPU 版本 | GPU 版本 | 说明 |
|------|---------|---------|------|
| YOLOv8 推理 | ~200ms/帧 | ~30ms/帧 | CPU 慢但够用 |
| MediaPipe 姿态检测 | ~50ms/帧 | ~20ms/帧 | CPU 可接受 |
| 模型训练 | ❌ 不推荐 | ✅ 推荐 | CPU 太慢 |

**建议**: 
- ✅ **开发和测试**: CPU 版本完全够用
- ⚠️ **生产部署**: 如有大量实时推理需求，考虑 GPU
