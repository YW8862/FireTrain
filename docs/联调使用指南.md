## 🧪 手动测试 API

### 使用 cURL 测试

#### 1. 用户注册

```bash
curl -X POST http://localhost:8000/api/user/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "student001",
    "email": "student001@firetrain.com",
    "password": "Test123456",
    "phone": "13800138001"
  }'
```

#### 2. 用户登录

```bash
curl -X POST http://localhost:8000/api/user/login \
  -d "username=student001&password=Test123456"
```

响应示例：
```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "token_type": "bearer",
  "user_info": {...}
}
```

#### 3. 获取用户信息（需要 Token）

```bash
# 将 YOUR_TOKEN 替换为实际 token
curl -X GET http://localhost:8000/api/user/profile \
  -H "Authorization: Bearer YOUR_TOKEN"
```

#### 4. 开始训练

```bash
curl -X POST http://localhost:8000/api/training/start \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "training_type": "extinguisher",
    "duration_seconds": 60
  }'
```

#### 5. 获取训练历史

```bash
curl -X GET "http://localhost:8000/api/training/history?page=1&page_size=10" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

#### 6. 获取个人统计

```bash
curl -X GET http://localhost:8000/api/stats/personal \
  -H "Authorization: Bearer YOUR_TOKEN"
```

#### 7. 获取训练趋势

```bash
curl -X GET "http://localhost:8000/api/stats/trend?days=7" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 使用 Python 脚本测试

```python
import requests

BASE_URL = "http://localhost:8000"

# 1. 登录
response = requests.post(f"{BASE_URL}/api/user/login", 
                        data={"username": "student001", "password": "Test123456"})
token = response.json()["token"]
print("Token:", token)

# 2. 获取用户信息
headers = {"Authorization": f"Bearer {token}"}
response = requests.get(f"{BASE_URL}/api/user/profile", headers=headers)
print("用户信息:", response.json())

# 3. 开始训练
response = requests.post(f"{BASE_URL}/api/training/start", 
                        headers=headers,
                        json={"training_type": "extinguisher", "duration_seconds": 60})
print("训练创建:", response.json())

# 4. 获取训练历史
response = requests.get(f"{BASE_URL}/api/training/history", 
                       headers=headers,
                       params={"page": 1, "page_size": 10})
print("训练历史:", response.json())

# 5. 获取统计
response = requests.get(f"{BASE_URL}/api/stats/personal", headers=headers)
print("个人统计:", response.json())
```

## 🛠️ 常见问题排查

### 问题 1: 服务无法启动

**症状**: `make docker-up` 失败

**解决方案**:
```bash
# 1. 检查端口占用
lsof -i :8000
lsof -i :5173
lsof -i :3306

# 2. 释放端口
kill -SIGTERM <PID>

# 3. 清理 Docker 资源
docker compose down
docker compose up -d
```

### 问题 2: Token 鉴权失败

**症状**: 
```json
{
  "detail": "无法验证凭据",
  "code": 401
}
```

**解决方案**:
1. 确认请求头包含 `Authorization: Bearer <token>`
2. 确认 Token 未过期（有效期 30 分钟）
3. 重新登录获取新 Token

### 问题 3: 数据库连接失败

**症状**: SQLAlchemy 数据库连接错误

**解决方案**:
```bash
# 1. 检查 MySQL 服务状态
docker compose ps mysql

# 2. 查看 MySQL 日志
docker compose logs mysql

# 3. 重启服务
make docker-restart
```

### 问题 4: API 接口返回 500 错误

**症状**: 服务器内部错误

**解决方案**:
```bash
# 1. 查看详细错误日志
docker compose logs backend | grep ERROR

# 2. 检查请求参数是否正确
# 3. 查看后端代码是否有异常
```
