## 🎯 实现功能

### 1. 开始训练任务 ✅
- 创建训练记录（状态：created）
- 记录训练类型、预计时长
- 自动设置开始时间
- 返回训练 ID 用于后续操作

### 2. 上传视频 ✅
- **方式一**：上传文件路径（适合已有文件）
- **方式二**：上传真实文件（保存到服务器）
- 状态流转：created → processing
- 验证训练状态和用户权限

### 3. 完成训练并评分 ✅
- 当前使用模拟分数（AI 评分待后续接入）
- 生成步骤分数（JSON 格式）
- 计算总分（平均分）
- 生成反馈文本和建议
- 状态流转：processing → done

### 4. 查询训练详情 ✅
- 获取单条训练记录的完整信息
- 包含基本信息、分数、反馈
- 验证用户权限（只能查看自己的记录）

### 5. 查询训练历史 ✅
- 分页查询（默认 10 条/页）
- 条件筛选：
  - 按状态（created/processing/done）
  - 按日期范围（start_date/end_date）
- 返回总数、当前页、每页数量
- 按创建时间倒序排列

### 6. 状态管理 ✅
- **created**: 训练已创建，等待上传视频
- **processing**: 视频已上传，等待评分
- **done**: 训练已完成，评分已保存

### 7. 模拟评分系统 ✅
- 生成 3 个步骤的分数
- 每个步骤包含：
  - 步骤名称
  - 分数（0-100）
  - 是否正确
  - 反馈意见
- 根据总分生成不同等级的反馈：
  - ≥90: 优秀
  - ≥80: 良好
  - ≥60: 合格
  - <60: 需要加强

## 🔐 安全特性

### 用户权限验证
- 只能查看自己的训练记录
- 只能操作自己的训练记录
- 无权访问他人数据（返回 403 Forbidden）

### 状态验证
- 只能在 created 状态上传视频
- 只能在 created/processing 状态完成训练
- 防止状态混乱

### 数据验证
- 使用 Pydantic Schema 验证请求参数
- 分数范围验证（0-100）
- 分页参数验证（page_size 最大 50）

## 📝 API 使用示例

### 1. 开始训练
```bash
curl -X POST http://localhost:8000/api/training/start \
  -H "Content-Type: application/json" \
  -d '{
    "training_type": "extinguisher",
    "duration_seconds": 120
  }'
```

响应：
```json
{
  "training_id": 1,
  "status": "created",
  "message": "训练已创建，请上传视频"
}
```

### 2. 上传视频（路径方式）
```bash
curl -X POST http://localhost:8000/api/training/upload \
  -H "Content-Type: application/json" \
  -d '{
    "training_id": 1,
    "video_path": "/path/to/video.mp4"
  }'
```

### 3. 完成训练并评分
```bash
curl -X POST http://localhost:8000/api/training/complete/1
```

响应：
```json
{
  "message": "训练已完成",
  "training_id": 1,
  "status": "done",
  "total_score": 85.67,
  "feedback": "良好！基本掌握操作要领",
  "scoring_details": {
    "total_score": 85.67,
    "step_scores": {
      "step1": {...},
      "step2": {...},
      "step3": {...}
    },
    "feedback": "良好！基本掌握操作要领",
    "suggestions": [...]
  }
}
```

### 4. 查询训练详情
```bash
curl -X GET http://localhost:8000/api/training/1
```

### 5. 查询训练历史
```bash
# 获取所有记录（分页）
curl -X GET "http://localhost:8000/api/training/history?page=1&page_size=10"

# 按状态筛选
curl -X GET "http://localhost:8000/api/training/history?status=done"

# 按日期范围筛选
curl -X GET "http://localhost:8000/api/training/history?start_date=2026-03-01&end_date=2026-03-12"
```
