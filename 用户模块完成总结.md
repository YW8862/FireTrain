## 🎯 实现功能

### 1. 用户注册 ✅
- 支持用户名和邮箱注册
- 密码 bcrypt 加密存储
- 检查用户名/邮箱重复
- 返回注册成功消息和用户 ID

### 2. 用户登录 ✅
- 支持用户名或邮箱登录
- 密码验证
- 生成 JWT Token（30 分钟有效期）
- 返回 Token 和完整用户信息
- 更新最后登录时间

### 3. 查看个人信息 ✅
- 需要 Bearer Token 认证
- 从 JWT Token 中解析用户 ID
- 从数据库查询真实用户数据
- 返回完整用户信息

### 4. 修改个人信息 ✅
- 需要 Bearer Token 认证
- 支持更新昵称、手机号、头像 URL
- 实时更新数据库
- 返回更新后的用户信息

### 5. 退出登录 ✅
- 需要 Bearer Token 认证
- 将 Token 加入黑名单
- Token 立即失效
- 内存实现（生产环境建议用 Redis）

### 6. 异常处理 ✅
- 重复用户名：400 Bad Request
- 重复邮箱：400 Bad Request
- 密码错误：401 Unauthorized
- 用户不存在：404 Not Found
- Token 无效：401 Unauthorized
- Token 已失效（黑名单）：401 Unauthorized

## 🔐 安全技术细节

### 密码加密
- **算法**: bcrypt
- **Salt Rounds**: 12
- **存储格式**: `$2b$12$...` (60 字符)

### JWT Token
- **算法**: HS256
- **有效期**: 30 分钟（可配置）
- **Payload**: 
  - `sub`: 用户名
  - `user_id`: 用户 ID
  - `exp`: 过期时间戳

### Token 黑名单
- **实现**: 内存存储（字典 + 集合）
- **自动清理**: 过期 token 自动删除
- **即时生效**: 注销后 token 立即失效

## 📝 使用示例

### 1. 用户注册
```bash
curl -X POST http://localhost:8000/api/user/register \
  -H "Content-Type: application/json" \
  -d '{
    "username": "newuser",
    "email": "user@example.com",
    "password": "secure123",
    "phone": "13800138000"
  }'
```

### 2. 用户登录
```bash
curl -X POST http://localhost:8000/api/user/login \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "username=newuser&password=secure123"
```

### 3. 获取个人信息
```bash
curl -X GET http://localhost:8000/api/user/profile \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```

### 4. 更新个人信息
```bash
curl -X PUT http://localhost:8000/api/user/profile \
  -H "Authorization: Bearer YOUR_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{"phone": "13900139000"}'
```

### 5. 退出登录
```bash
curl -X POST http://localhost:8000/api/user/logout \
  -H "Authorization: Bearer YOUR_TOKEN_HERE"
```
